# Safal Gautam Portfolio 

**A full-stack portfolio showcasing my tech stack and projects.**  
Built with **React** on the frontend and **Node.js** on the backend, it dynamically fetches data from the **GitHub API** and other tools and also manages blogs via markdown files in the database.


## Features

- **Full-Stack Architecture** – React frontend + Node.js backend with Blog Management System.  
- **GitHub Projects Integration** – Fetches repositories dynamically, filters, sorts, and displays them as interactive project cards.  
- **Tech Stack / Skills Section** – Highlights my core technologies: JavaScript, React, Node.js, Python, C/C++, HTML, CSS, TypeScript, Jupyter.  
- **Blog Management System** – Upload, fetch, and manage blogs stored as markdown files in the database, with dynamic rendering.  
- **Responsive & Interactive UI** – Hover effects, scaling animations, and mobile-friendly sidebar navigation.  
- **Downloadable CV** – Users can download my CV directly from the portfolio.  


## Tech Stack

- **Frontend:** React, Tailwind CSS, React Router, Axios  
- **Backend:** Node.js, Express  
- **Blog Storage:** MongoDB, Markdown files in database (MongoDB / JSON)  
- **Data Integration:** GitHub API for dynamic project fetching  
- **Icons & Assets:** react-icons, SVGs  
- **Tools:** Git, GitHub, VS Code  


##  How to use ?

Clone the repository, install dependencies, start the backend & frontend dev server 
```bash
git clone 
https://github.com/safalgautam22/safalgautam.com.np.git

cd client
npm install

cd ../server
npm install

npm run start
cd ../client

npm run dev
```

Now open app in your browser at `http://localhost:5173`
